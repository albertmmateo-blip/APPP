## Quick Start

1. Select a category tab (Trucar, Encarregar, Factures, or Notes)
2. Tap the **+** button to create a new note
3. Fill in the note details (Name and Body are required)
4. Tap **SAVE**

Your note is now saved and will appear in the selected category!
